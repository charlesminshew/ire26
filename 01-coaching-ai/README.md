# Session 1 — Coaching AI Tools for Investigative Reporting

**Friday 2:45 PM · 1 hour · Maryland 3-4**

AI tools hallucinate, hedge, and hedge wrong. This session teaches you how to push back — how to write prompts that get specific, useful outputs for investigative work: FOIA letters, data dictionaries, spreadsheet formulas, and debugging code.

## What you'll learn

- How to write prompts that constrain AI output to what you actually need
- Patterns for FOIA and public-records requests
- Using AI to decode data dictionaries and agency jargon
- Getting AI to write and explain spreadsheet formulas
- Debugging code with AI step by step

## What to bring

- Laptop (no tablets)
- Free account at [ChatGPT](https://chat.openai.com) or [Claude](https://claude.ai)
- A real investigative problem you're stuck on (optional but useful)
