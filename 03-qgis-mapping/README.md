# Session 3 — Master the Power of Mapping with QGIS

**Saturday 2:30 PM · 3 hours 30 minutes · Maryland 5-6**

With Jennifer LaFleur. A hands-on workshop on using QGIS for investigative mapping: loading data, joining tables, styling maps, and producing publication-ready output.

> **Limited seating — pre-registration required.**

## Install QGIS before the session

Install **QGIS 3.44.10 Solothurn LTR** — do not install the latest release.

- [Download QGIS](https://qgis.org/en/site/forusers/download.html) — choose the **Long Term Release (LTR)** version
- Windows: use the standalone installer
- Mac: use the official .dmg
- Linux: use the package manager instructions on the download page

Verify the version number says **3.44.10** before the session.

## What you'll learn

- Loading shapefiles, GeoJSON, and CSV data with coordinates
- Joining spreadsheet data to geographic boundaries
- Styling choropleth and point maps
- Filtering and selecting features by attribute
- Exporting maps for print and web
